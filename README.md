## 🚀 快速开始

```bash
git clone https://github.com/huanghhcri/ai-llm-tutorial.git
cd ai-llm-tutorial
pip install mkdocs mkdocs-material
mkdocs serve
# 访问 http://127.0.0.1:8000
```

## 📁 目录结构


## ✨ 特性

- 🎯 **分类导航**：顶部标签页切换不同系列
- 📝 **批注笔记**：选中文字即可添加个人批注（Hypothesis）
- 🔍 **全文搜索**：支持中文搜索建议与高亮
- 🌙 **暗色模式**：一键切换浅色/暗色主题
- 📱 **响应式**：桌面端与移动端均可用
